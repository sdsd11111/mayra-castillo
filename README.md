# Sitio Web - Ing. Mayra Castillo

## Descripción

Sitio web profesional para la Ingeniera Mayra Castillo, contadora especializada en servicios tributarios, comercio exterior, formalización de negocios y trámites digitales.

## Características

- **Diseño Responsive**: Compatible con dispositivos móviles y de escritorio
- **Múltiples Páginas**: Navegación completa entre diferentes secciones
- **Formulario de Contacto**: Integrado con WhatsApp para comunicación directa
- **Páginas de Servicios**: Información detallada de cada servicio ofrecido
- **Diseño Profesional**: Colores corporativos y tipografía moderna

## Estructura del Sitio

### Páginas Principales
- **Inicio** (`/`): Página principal con información general y servicios
- **Nosotros** (`/nosotros`): Historia, misión, visión y valores
- **Contacto** (`/contacto`): Formulario de contacto e información de ubicación

### Páginas de Servicios
- **Trámites Tributarios** (`/servicios/tributarios`): Declaraciones y asesoría fiscal
- **Comercio Exterior** (`/servicios/comercio-exterior`): Importaciones y exportaciones
- **Formalización de Negocios** (`/servicios/formalizacion`): Constitución de empresas
- **Trámites Digitales** (`/servicios/digitales`): Firma electrónica y gestiones online

## Tecnologías Utilizadas

- **React 18**: Framework de JavaScript para la interfaz de usuario
- **React Router**: Navegación entre páginas
- **Vite**: Herramienta de desarrollo y construcción
- **CSS3**: Estilos personalizados con diseño responsive
- **Lucide React**: Iconos modernos y profesionales

## Instalación y Uso

### Prerrequisitos
- Node.js (versión 16 o superior)
- npm o yarn

### Instalación
```bash
# Navegar al directorio del proyecto
cd mayra-castillo-website

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# El sitio estará disponible en http://localhost:5173
```

### Construcción para Producción
```bash
# Generar archivos optimizados para producción
npm run build

# Los archivos se generarán en la carpeta 'dist'
```

### Vista Previa de Producción
```bash
# Previsualizar la versión de producción
npm run preview
```
